# Sample Data

This folder contains example datasets for testing RivaPlan.

Study area: Rapti River

Coordinate system: EPSG:32644

Datasets:
- Left bank (1989)
- Right bank (1989)
- Left bank (2025)
- Right bank (2025)

These files can be used to test:
- Bank lines to Centerline
- Width
- Sinuosity
- Curvature
- Centerline Migration
- Bankline Migration
